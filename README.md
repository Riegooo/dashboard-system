#  Dashboard System (Node.js CLI App)

A **prompt-based / command-line dashboard system** built using **Node.js**.

---

##  Features

- User authentication system
- Role-based access control
- admin / Moderator / Guest roles
- Menu-driven dashboard navigation
- Modular and maintainable code structure
- Utility and helper functions

---

##  NPM Script Setup

Add the following script to your `package.json`:

```json
"scripts": {
  "start": "node index.js"
}
```

## How to Run

### Clone this project (if downloading from GitHub)

```bash
git clone https://github.com/Riegooo/dashboard-system.git
cd dashboard-system
```
install dependencies
```bash
npm install
```
Start the application
```bash
npm start
```
